Interaction model
  • Knob — drag vertically (↑ = higher pitch). Click to reset to 0.
  • Range pills — set semitone ceiling (±1 / ±6 / ±12 / ±24 st). Switches knob sensitivity without changing pitch.
  • Fine tune — ±100 cents, same commit-on-release pattern as other FX sliders.
  • Processing fires on pointerup / keyup only (no live reprocessing).
Implementation note
  • Uses OfflineAudioContext + phase-vocoder pitch shift (or playbackRate approximation for MVP).
  • New FxType = 'pitch', params: { semitones: number; cents: number; range: 1|6|12|24 }
  • Knob SVG arcs recalculated reactively from semitones / range ratio.
