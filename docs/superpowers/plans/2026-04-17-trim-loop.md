# Trim Loop Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the pause→restart loop mechanism in WaveformEditor with a `requestAnimationFrame` poll that seeks back to `region.start` without ever pausing, eliminating the audible gap at every loop boundary.

**Architecture:** A `loopRafRef` holds the active rAF handle. `startLoopRaf` launches a tick function that runs every ~16 ms while WaveSurfer is playing; when `currentTime >= region.end` it calls `ws.setTime(region.start)` (no stop/start). `stopLoopRaf` cancels the frame. All `region.play()` calls are replaced with plain `ws.play()` + `startLoopRaf()` so WaveSurfer never schedules its own internal pause-at-end.

**Tech Stack:** React 18, WaveSurfer.js v7, TypeScript, Web Audio API (`requestAnimationFrame`)

  ```ts
      ws.play();
      startLoopRaf();
      return;
    }
    stopLoopRaf();
    onPlayingChangeRef.current(false);
  });
  ```
