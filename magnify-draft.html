Triggers (same 500 ms hold threshold)
  • Waveform canvas hold — magnifies the region around the held position for cursor-based fine scrubbing.
  • Trim handle hold / drag — magnifies the region around that handle's current position; dragging inside the magnified view moves the handle at sub-pixel precision. The caret / panel label indicates which handle is active (trim start or trim end).
Shared behaviour
  • Zoomed window = ±250 ms centred on the hold point.
  • Dragging within the zoomed view moves the handle; the main waveform handle moves in sync.
  • Release closes magnification.
  • The active time label in the main view turns white + bold while magnified.
  • A brief pulsing ring around the handle signals zoom activation.
